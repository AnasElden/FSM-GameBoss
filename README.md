# Boss AI — FSM Simulator 

Interactive web app for the paper:

> **"Graph-Based Discrete Modeling of an Adaptive Game Boss"**  
> Youssef Mohamed Abdelatif & Anas Abduallah Mostafa  
> MSA University, Giza, Egypt

## Features

- Live FSM state transitions with HP bar
- Manual action buttons (melee, ranged, magic, dodge, block, idle)
- Auto-replay of exact paper traces (Table 5 & Table 6)
- Full encounter log with δ output column
- Adjacency matrix (Table 2)
- Markov transition matrix (Table 4)
- Transition function table (Table 3)
- Complexity analysis (Section 8.1)
- State path visualization
